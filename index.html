Standard 3-Tier Subscription Model

A subscription structure designed for organizations running fundraising and capital campaigns, from early-stage nonprofits to large institutions.

Basic

Starter

Essential tools for presenting fundraising projects.

Features

  • ✔ 2D plans and basic 3D viewing
  • ✔ Project descriptions, campaign goals and milestones
  • ✔ Viewing Naming Opportunities (read-only)
  • ✔ Static images and media galleries
  • ✔ Up to 3 active projects
  • ✔ Giftabulator (same unified tool across all plans)
  • ✖ No AI features
  • ✖ No Unreal Engine environments

Best for

Small nonprofits, early-stage campaigns, and introductory donor presentations.

Pro

Most Popular

Advanced visualization and donor engagement capabilities.

Features

  • ✔ Full 3D navigation and 360° environments
  • ✔ Interactive Naming Opportunities (booking, status tracking, notifications)
  • ✔ Giftabulator (same unified tool as Basic and Enterprise)
  • ✔ AI Assistant: auto-generated project descriptions, donor-friendly explanations of project zones, and tailored donor suggestions
  • ✔ Presentation modes for major donor meetings
  • ✔ Static Unreal Engine scenes (renders and panoramas)
  • ✔ Up to 20 active projects

Best for

Growing organizations, schools, hospitals, museums, and mid-to-large fundraising campaigns.

Enterprise

For Large Campaigns

Immersive, high-performance environment for large-scale fundraising.

Features

  • ✔ Advanced interactive Unreal Engine spaces (animated, real-time, interactive)
  • ✔ Full AI Suite: giving forecast models, donor-scenario simulations, and personalized donor journeys
  • ✔ Giftabulator (same unified tool across all plans)
  • ✔ API access, integrations, and white-label options
  • ✔ Unlimited projects
  • ✔ VR / immersive presentations for capital campaigns

Best for

Universities, major foundations, large development projects, and multi-million fundraising campaigns.
